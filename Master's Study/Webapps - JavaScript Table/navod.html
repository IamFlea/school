Tento dokument popisuje použití implementovaného řešení. Pro úspěšné použití do hlavičky HTML dokumentu vložte kód pro nahrání přiloženého JavaScriptu proj.js 
<script type="text/javascript" src="proj.js"></script>
Dynamickou tabulku table opatřete specifickým atributem id, například 
<table id="dynamic">
Tělíčko body obohaťte o atribut onload s hodnotou dynamicTable('id_tabulky');. Například 
<body onload="dynamicTable('dynamic');">
Implicitní řazení tabulky je podle znaků lokální abecedy. Pro řazení podle numerických hodnot přidejte atribut class s hodntou numeric. Tuto úpravu proveďte v prním řádku tabulky, kde hodnoty jsou buňěk jsou ve formátu <th>, daného sloupce. Například: 
<table>
    <tr>
        <th class="numerical">ID</th>
        <th>Species</th>
        <th class="numerical">Age</th>
        <th class="alphabetical">Free time activitz</th>
        <th class="alphabetical">Favourite food</th>
        <th class="numerical">Brain cells</th>
    </tr>
    ...
Položky v <thead> a <tfoot> jsou při řazení ignorovány.

Skript rovněž podporuje vlastní nastavní stylů například podle souboru proj.css. Nově přidané buňky obsahují třídu class s hodnotou dynamic_header_colN, kde N značí pořadí sloupce. V css souboru s ním můžete různě manipulovat, např. 
.dynamic_header_col0{
	background-color: yellow;
}
Všechna textová pole mají třídu s hodnotou dynamic_textfield, "ikonky" řazení mají rovněž svou specifickou třídu dynamic_ASCfield, resp. dynamic_DESCfield. Příklad CSS souboru tedy může být následující: 
.dynamic_textfield {
	color: orange;
}

.dynamic_ASCfield{
	color: red;
	font-family: sans-serif;
}

.dynamic_DESCfield{
	color: green;
	font-family: sans-serif;
}
Příklady použití lze spatřit zde. Projekt byl úspěšně testován na prohlížečích Firefox, Chrome a Internet Explorer v aktuálních verzích.
